Como crear nuestra primera remesa

En la pantalla principal se distinguen dos partes: a la izquierda los botones de la aplicación y a la derecha la lista de remesas ya introducidas.




Para crear una remesa de recibos pulsaremos sobre , tras lo cual nos pedirá un nombre de remesa.



Una vez introducida la nueva remesa pasamos a la pantalla de la remesa.



En esta pantalla lo primero que hemos de hacer es seleccionar el ordenante. Hasta que no seleccionemos el ordenante no nos permitira añadir ningún recibo.



Para dar de alta un ordenante se deben de rellenar todos los datos. El unico dato que quizas pueda dar algún problema es el del sufijo.
El sufijo es un código de 3 números que nos proporcionaría la entidad bancaria. El sufijo se utiliza para identificar a un mismo ordenante con distintos numero de cuenta.
Por ejemplo:
Suponemos un ordenante con NIF B12345678. Este ordenante envia remesas por dos concepto diferentes y necesita que se abonen en cuentas diferentes con lo que utiliza el sufijo 001 para la cuenta 1 y el sufijo 002 para la cuenta 2:
Cuenta 1 --> B12345678-001
Cuenta 2 --> B12345678-002

Una vez que hemos rellenado todos los datos del ordenante pulsaremos sobre el botón de Añadir.
Para seleccionar ese ordenante haremos doble click sobre el y despues saldremos de esta pantalla.



Una vez que hemos seleccionado el ordenante iremos a añadir el primer recibo. Para hacerlo pulsaremos sobre el botón de Añadir en la pantalla de remesas y pasaremos a la ventana de recibos.



En esta pantalla seleccionaremos el cliente pulsando sobre el botón de Buscar.



Después de seleccionar el cliente introduciremos el importe y los conceptos que queramos para este recibo y pulsaremos para grabar el recibo.



Iremos introduciendo todos los recibos de la remesa y una vez que la finalicemos pulsaremos sobre el botón de generar para grabar la remesa.
